Getting Started

Welcome to ALM Karvaan! This guide will help you get up and running with our audio intelligence platform. Whether you're a developer integrating our API or a user exploring our web interface, you'll find everything you need here.

Prerequisites

  • A valid ALM Karvaan account
  • API credentials (for developers)
  • Modern web browser or Python 3.8+

Installation

Web Interface

Simply navigate to app.almkarvaan.com and log in with your credentials. No installation required!

Python SDK

Install the ALM Karvaan Python SDK:

pip install almkarvaan

API Reference

Audio Analysis Endpoint

Analyze audio files for noise, emotion, and environmental context.

POST /api/v1/analyze
Content-Type: multipart/form-data

Parameters:
- audio_file: Audio file (MP3, WAV, FLAC)
- features: ['noise', 'emotion', 'environment']

Response Format

{
"status": "success",
"analysis": {
"noise_level": 45,
"emotion": "neutral",
"environment": "office"
},
"timestamp": "2026-01-30T10:00:00Z"
}

Examples

Python Example

import almkarvaan

client = almkarvaan.Client(api_key="YOUR_API_KEY")
result = client.analyze_audio("sample.mp3")
print(result.noise_level)
print(result.emotion)

JavaScript Example

const alm = require('almkarvaan');

const result = await alm.analyzeAudio('sample.mp3');
console.log(result.noiseLevel);
console.log(result.emotion);

Frequently Asked Questions

What audio formats are supported?

We support MP3, WAV, FLAC, OGG, and AAC formats. Maximum file size is 500MB.

How long does analysis take?

Analysis typically completes in 5-30 seconds depending on audio length and selected features.

Is my audio data secure?

Yes! All audio data is encrypted in transit and at rest. We never store or share your data with third parties.
